# About the project

Welcome to my portfolio. This app was built with [Next.js](https://nextjs.org/) showcasing my projects, skills, experience and a bit about me. It was designed in mind of SEO optimizations and a modern look.

## Built with

- [Typescript](https://www.typescriptlang.org/) Used for type safety
- [Next.js](https://nextjs.org/) Core framework used primarily for SSG and SEO rendering
- [Tailwind CSS](https://tailwindcss.com/) CSS framework used for fast UI styling
- [Framer motion](https://motion.dev/) Used for smooth animations/transition to enhance UX
- [Next-intl](https://next-intl.dev/) Enables internationalization (i18n), making site ready in English and Polish
- [Next-themes](https://github.com/pacocoursey/next-themes) Provides easy support for light and dark mode

## Getting Started

```bash
# Clone the repo
git clone https://github.com/MadGotten/portfolio.git

# Navigate into the project
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```
